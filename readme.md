# 台北市南港、松山、內湖公園資訊

為了帶小孩出去玩不要一直 google 找地點，就串接台北市政府 open data 資料當作自己的公園資料。

## 使用技術
* open data 資料串接。
* jQuery 互動效果。
* 使用 JavaScript 判斷透過 select 取得對應資料。
* 自行撰寫 RWD。
* 使用 SCSS 預處理器管理 CSS。
